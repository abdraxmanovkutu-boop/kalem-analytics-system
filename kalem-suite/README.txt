Kalem Suite — демо-проект для диплома

Что внутри:
1) index.html — публичный сайт издательства
2) styles.css + script.js — дизайн и аналитика
3) admin/login.html — скрытый вход в админку
4) admin/dashboard.html — аналитическая панель
5) admin/admin.css + admin/admin.js — стили и логика админки

Как использовать:
1. Открой index.html
2. Переходи по карточкам, нажимай кнопки, заполняй опрос
3. Затем открой admin/login.html
4. Логин: admin
5. Пароль: Kalem2026!

Важно:
Это статическая demo-версия на HTML/CSS/JS.
Для настоящей приватной аналитики нужен backend (например Node.js + Express) и база данных.
